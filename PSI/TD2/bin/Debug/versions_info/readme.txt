Les données pour une version N sont données sur la ligne N du fichier
La ligne 0 ne contient les données d'aucune version de code QR.

On trouve sur la ligne, dans l'ordre, séparés par des espaces
- La capacité en caractères alphanumériques
- Le nombre de mots de données requis
- Le nombre de mots de correction
- Le nombre de blocs du groupe 1
- Le nombre de mots dans un bloc du groupe 1
- Le nombre de blocs du groupe 2
- Le nombre de mots dans un bloc du groupe 2
- Les positions des ALIGNEMENT PATTERN, séparées par des virgules
